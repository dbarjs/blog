Novo blog!

É com muito orgulho que inicio aqui a nova era do Noibe com blogs pessoais! E este é o primeiro post! :)

Estrutura

Nessa nova estrutura de blog, eu idealizei um Framework Web chamada Lingvo - sim, isso é um 'v' - focada, inicialmente, em quem entende o básico de desenvolvimento Web. O objetivo por enquanto é liberdade na publicação de conteúdos tomando controle de tudo o que acontece na publicação do arquivo.

Estarei usando no inicio do projeto Lingvo uma estrutura bastante "rudimentar" que lembra o a Web de antigamente utilizando pastas para armazenar e publicar o conteúdo. Sim, eu sei que existem vários frameworks ou padrões prontos para isso, mas agora tudo o que eu quero é simplicidade, performance e segurança de que o conteúdo será visualizado por qualquer cliente (browser). No momento eu irei utlizar padrões persistentes na exibição e vou evitar usar qualquer linguagem serverside para fazer o controle de conteúdo. Vou utlizar, no entanto, uma API já pronta para capturar e ter controle dos acessos na página. O esquema de links também será feito manualmente. Vou utilizar apenas o Villa Framework para me auxiliar no desenvolvimento.

Gente, tudo o que preciso agora é de simplicidade! Em breve estarei reunindo o pessoal da comunidade para desenvolver algo incrível! :)

Marcação

A marcação será utilizada no padrão HTML5 sendo composta pelas tags header, main, article, section e footer com base no artigo do Tree House sobre elementos de sessão em HTML5.

Tipografia

Estou estudando pelo menos 10 minutos todos os dias sobre o fantástico mundo da tipográfia para entregar aqui a melhor tipográfia possível para todos os públicos.

Cores

Estarei utilizando as cores padrões do Material Design com um cinza 90% (#212121) como padrão nos textos e fundo completamente branco (#ffffff).

Espaçamento

Ainda não defini uma boa estrutura de espaçamentos (tamanho de linhas e espaço de paragrafos), nas próximas versões do Villa virão melhorias para isso.

Tamanho

Inicialmente estarei entregando um tamanho de 16px (12pt) com titulos de tamanho aleatório. Os tamanhos sempre serão baseados no padrão do Villa Framework, este padrão foi criado para a utilização do métrica tipografica "em".

Container

No espaço de conteúdo (container), estarei utlizando o padrão do Villa Framework.

Frequencia de escrita

Estarei tentando escrever aqui todos os dias com alguma coisa aleatória sobre desenvolvimento, origem da vida e tudo mais.

Atualizações

Para qualquer alteração de padrão no framework Lingvo eu irei manter tudo nessa página, todo o controle de versionamento estará disponível no repositório do projeto localizado no Github que pode ser acessado clicando aqui.

Pra terminar

Por enquanto é isso, estarei criando aqui em breve um post sobre a minha aprendizagem com Regex.
